# Google Assistant・IFTTT・AWS・Raspberry Pi でモールス信号の生成

## Google Assistant に音声を認識させ、IFTTT で webhooks を経由して AWS サーバーに POST それをさらに Raspberry Pi に送り LED ライトで点滅させることでモールス信号を作る
